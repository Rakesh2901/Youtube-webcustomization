# ✨ Youtube-webcustomization
### Glassmorphism • Smooth Animations • Clean UX

## 🧊 Overview

A **high-performance YouTube UI customization** that transforms the default interface into a **modern Apple-inspired experience** using:

- Glassmorphism (blur + depth)
- Smooth micro-interactions
- Clean typography & spacing
- Refined hover and focus states

Built on top of **Enhancer for YouTube**, this project keeps everything lightweight and non-intrusive.

---
## ⚙️ Installation Steps

1. Install **Enhancer for YouTube** extension  
2. Open extension settings  
3. Go to:
   - **Custom CSS**
   - **Custom Script**
4. Paste:
   - `styles.css` → into Custom CSS  
   - `script.js` → into Custom Script  
5. Reload YouTube  

✅ Done — Enjoy the new UI

## ✨ Features

- 🧊 **Glassmorphism UI** (blur, transparency, depth)
- ⚡ **Smooth animations** (hover, focus, transitions)
- 🎯 **Single highlight system** (no double effects)
- 🎨 **Clean tags & chips** (fixed broken UI)
- 🔍 **Working search bar blur + tooltip**
- 🎬 **Stable video player UI**
  - No edge glow
  - Consistent icon size
- 🧩 Modular and easy to customize
